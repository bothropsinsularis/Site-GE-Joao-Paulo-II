## 1. Adentro
#### 1.1. Sobre a Plataforma
___

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) para o ensino médio integrado ao técnico em informática para internet na ETEC de Campo Limpo Paulista. Ele aborda o desenvolvimento de uma plataforma digital para o Grupo Escoteiro João Paulo II, com o objetivo de facilitar a divulgação das atividades do grupo e promover a comunicação entre os membros, respondendo à necessidade de melhorar o engajamento e a participação nas ações escoteiras. A plataforma visa conectar os escoteiros, facilitar o planejamento de eventos e fortalecer a interação entre os membros e a comunidade.

#### 1.2. Motivação
___

A motivação para o desenvolvimento deste projeto surgiu da necessidade de criar uma solução eficiente para a divulgação das atividades do Grupo Escoteiro João Paulo II e para facilitar a comunicação entre os membros. A dificuldade em compartilhar informações sobre eventos, ações e encontros pode retardar a participação e o engajamento dos escoteiros, limitando o potencial de crescimento do grupo. A plataforma tem como objetivo preencher essa lacuna, proporcionando uma forma prática e acessível de manter todos os membros informados e conectados, fortalecendo a interação e o envolvimento nas atividades escoteiras.

<br>


#### 1.3. Objetivos
___
Foram determinados os seguintes objetivos para a plataforma de divulgação do Grupo Escoteiro João Paulo II:

- **Desenvolver** uma plataforma web: Criar um ambiente digital que reflita os valores e objetivos do movimento escoteiro, oferecendo funcionalidades práticas e acessíveis para divulgar as atividades do grupo e facilitar a interação entre os membros.

- **Identificar** as necessidades do público-alvo: Realizar pesquisas para compreender as principais dificuldades enfrentadas pelos escoteiros e suas famílias, permitindo que a plataforma seja projetada para atender de forma eficaz a essas necessidades.

- **Facilitar** a comunicação e a organização das atividades: Proporcionar ferramentas para divulgação de eventos, agendamento de reuniões e interação entre os membros, fortalecendo a participação nas atividades do grupo e ampliando o alcance do movimento escoteiro na comunidade local.

#### 1.2. Pré-Requisitos
___

Os pré-requisitos para a inicialização do sistema são:

- Um computador com o sistema operacional [Windows](https://www.microsoft.com/pt-br/windows?r=1) 7+;
- [XAMPP](https://www.apachefriends.org) 8+;
- Qualquer [IDE](https://www.redhat.com/pt-br/topics/middleware/what-is-ide#:~:text=IDEs%20mais%20usados-,IDE%20(Integrated%20Development%20Environment),de%20usuário%20gráfica%20(GUI).);
- Acesso à Internet*;
- Qualquer [navegador web](https://pt.wikipedia.org/wiki/Navegador_web).

> \* O acesso à internet só é necessário durante a fase de instalação.

<br>

#### 1.3. Instalação 
___
Clone/faça download deste repositório na sua pasta `/htdocs`. Inicie o Apache e MySQL na interface XAMPP. No navegador, acesse o endereço `localhost/phpmyadmin` e selecione `+ Novo` na seção de bancos de dados. Logo, entre na aba "Importar", e entre com o arquivo `jpii.sql`. De volta à uma nova guia do navegador, entre no endereço `localhost/jpii`.
